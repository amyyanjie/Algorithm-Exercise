## 题目

假设按照升序排序的数组在预先未知的某个点上进行了旋转。

( 例如，数组 [0,1,2,4,5,6,7] 可能变为 [4,5,6,7,0,1,2] )。

请找出其中最小的元素。

你可以假设数组中不存在重复元素。

**示例 1:**

```
输入: [3,4,5,1,2]
输出: 1
```

**示例 2:**

```
输入: [4,5,6,7,0,1,2]
输出: 0
```

## 题解

### 一、双指针

```java
    public int findMin(int[] nums) {
        int i = 0;
        int j = nums.length - 1;
        while (i != j) {
            if (nums[i] > nums[j]) {
                i++;
            } else {
                j--;
            }
        }
        return nums[j];
    }
```

* 时间复杂度：O(n)

## 二、二分法

**精简判断：**

* 当`num[mid]>=nums[right]`时，确定mid处一定非最小值，此时`left=mid+1`。

```java
    public int findMin(int[] nums) {
        int left = 0;
        int right = nums.length - 1;
        int mid;
        while (left != right) {
            mid = (left + right) / 2;
            if (nums[mid] < nums[right]) {
                right = mid;//mid可能是最小值
            } else {
                left = mid + 1;//mid一定不是最小值，注意需要+1，否则可能陷入循环
            }
        }
        return nums[left];
    }
```

* 时间复杂度：O(n)
